# gateways

**Backend**
* laravel 8
* php 8
* mysql

**Frontend**
* vuejs (3)
* quasar (3)
* typescript

> The front is now calling to the api that serves in localhost:8000, so to serve the backend is necessary to run the command

``` bash
php artisan serve
```

while for the front end

``` bash
quasar dev
```
